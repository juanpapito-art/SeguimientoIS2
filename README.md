## 📌 Sobre el Proyecto: PySL
**PySL** es una iniciativa tecnológica diseñada para optimizar y digitalizar las visitas de prevención y seguridad locativa realizadas por los cuerpos de bomberos en Colombia. 

El objetivo principal es transformar procesos manuales y lentos en un sistema dinámico y veloz que garantice la trazabilidad de la información y el cumplimiento de normativas vigentes como la **Ley 1575** y la **Ley 1801**.

### 🛠️ Fases del Proyecto
1. **Análisis:** Identificación de requerimientos técnicos.
2. **Diseño:** Arquitectura del sistema y bases de datos.
3. **Desarrollo:** Construcción del software (Frontend/Backend).
4. **Pruebas:** Verificación y puesta en marcha.
